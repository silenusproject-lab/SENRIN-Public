## 📱 SENRINについて

SENRINは、花火大会をARで楽しむための観測支援アプリです。

### 主な機能
- AR花火表示 - 打上地点の方向と花火の開花位置を3D空間で視覚化
- 最大15km先まで対応
- カメラ構図ガイド - 写真撮影に最適な構図をAR枠線で表示
- 多言語対応（日本語・英語）
